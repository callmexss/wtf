# What The F\*\*\* Does This Abbreviations Mean

A command line tool for looking up Acronyms & Abbreviations.

## Install

### setuptools

```sh
$ git@github.com:callmexss/wtf.git
$ pipenv shell
$ pipenv install
$ pip install --editable .
```
